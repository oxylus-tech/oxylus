Class List<M>

Handle a list of model instances fetched using Rest Api. It is used in model's panels.

It uses Query object in order to fetch items and relations.

Example

const value = ref(null)
const list = new List.reactive({
    query: new Query(repos.users, repos),
    relations: ['groups'],
    value,
})

await list.fetch({url: '/users'})
